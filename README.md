## الإعداد

### 1. إعداد Google OAuth
اتبع التعليمات في ملف `GOOGLE_SETUP.md` لإعداد تسجيل الدخول عبر Google.

### 2. ربط الباك إند
افتح ملف `api.js` وغير `API_BASE_URL` إلى رابط الباك إند الخاص بك:

```javascript
const API_BASE_URL = 'https://your-backend-url.com/api';
```

## الاستخدام

1. افتح `index.html` في المتصفح
2. سجل حساب جديد أو سجل دخول
3. تصفح الحرفيين أو أنشئ طلب خدمة
4. تواصل مع الحرفيين عبر المحادثات

## ملاحظات

- جميع البيانات حالياً تُحفظ في localStorage (للتطوير)
- في الإنتاج، يجب ربط الموقع بالباك إند الفعلي
- تأكد من تحديث `YOUR_GOOGLE_CLIENT_ID` في صفحات تسجيل الدخول
